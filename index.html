    How to make ssh key to your github

  • 1- open your gitbash and write this command [ ssh-keygen -t rsa -b 4096 -C "Your Email"]

    • Responce:- Generating public/private rsa key pair. Enter file in which to save the key (/c/Users/DELL/.ssh/id_rsa): Enter passphrase ( empty for no passphrase ):

  • 2- After Get the responce Write Your Password
  • 3- to check write this command cat ~/.ssh/id_rsa.pub
  • 4- take the resulte and go to github --> setting -->SSH and GPG Keys
  • 5- choose Add new ssh key & Put The Resulte & Press Enter
  • 6- Remember my-ssh Key Is M-E & M-P
